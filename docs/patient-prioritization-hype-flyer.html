What is Patient Prioritization?

For each admitted patient, CipherRounds assigns an advisory priority score (0–100) and a priority tag. Rounders sort and filter the worklist to focus on patients who need attention first.

Priority tags

Low0–59
Medium60–79
High80–100

How scoring works

Priority Score = weighted average of factor conversion scores. Each factor maps patient data to a 0–100 score via configurable rules, then weights (summing to 100) determine contribution to the final score.

  • Rule-based factors — LOS, age, admit source, rounding flags, admitting diagnosis, and more
  • Optional AI factor — Likelihood to be Dissatisfied predicts HCAHPS top-box risk; omit it to use rules only
  • Multi-program logic — when several programs match, the highest score wins
